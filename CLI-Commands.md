# PMM CLI Commands

## Normal workflow (for testing)

``` bash
cd ~/Desktop/PlayMyMusic
source venv/bin/activate
python3 main.py
```

==

## Enter PMM Project Folder

```bash
cd ~/Desktop/PlayMyMusic
```

## Activate Virtual Environment

```bash
source venv/bin/activate
```

## Run PMM

```bash
python3 main.py
```

## Exit Virtual Environment

```bash
deactivate
```

## Quick Launch (after creating helper script)

```bash
./run_pmm.sh
```

## Git Backup Workflow

```bash
git add .

git push
```

```bash
git status
```

## Pull Latest Github Version

```bash
git pull
```

```bash
chmod +x run_pmm.sh
```

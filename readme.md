
# PlayMyMusic (PMM)

A lightweight, local-first music player focused on simplicity, album art, and an always-current music library.

## Philosophy

PlayMyMusic reflects whatever music currently exists inside the user's Music folder.

No streaming.
No cloud services.
No recommendations.
No accounts.

Just local music playback with a clean, art-focused interface.

## Goals

* Scan Music folder on launch
* Play current folder contents
* Shuffle or normal playback
* Read embedded MP3 metadata
* Display album art
* Minimal, calm interface
* Smooth visualizer
* Linux-first development
